🏆 Gestión de Clubes Deportivos
Aplicación de escritorio desarrollada en C# con Windows Forms para la gestión de clubes, equipos, jugadores y entrenadores.

📌 Funcionalidades principales
Agregar jugadores y entrenadores

Crear equipos asignando entrenadores

Asociar jugadores a equipos

Listar participantes, entrenadores, equipos

Expulsar jugadores del sistema

Ver jugadores por equipo

🧠 Estructura del sistema
CJugador, CEntrenador, CEquipo: Clases que representan a los participantes.

CControladorTorneo: Maneja la lógica general del sistema.

Formularios como AgregarJugador, ListarParticipantes, VerJugadoresEquipos, etc., componen la interfaz.

⚙️ Tecnologías utilizadas
C#

Windows Forms (.NET Framework 4.7.2)

Visual Studio

🚀 Cómo ejecutar
Abrir el archivo .sln con Visual Studio

Compilar y ejecutar (F5)

Usar los formularios para cargar y consultar datos
